# 郭昱成｜UX/UI 作品集

这是一个纯静态 HTML 作品集网站，当前收录 Remi AI 记忆助手项目。

## 目录

```text
.
├── index.html              # 作品集首页
├── pages/                  # 案例页与可交互 Demo 页面
├── styles/                 # 全站公共样式
├── assets/remi/            # Remi 项目图片素材
├── demos/                  # 独立 Demo 源码
└── docs/                   # 作品集 PDF 等文档
```

## 页面入口

- `index.html`：项目目录
- `pages/case.html`：Remi AI 案例讲述
- `pages/remi.html`：Remi AI 可交互 Demo

## 本地预览

在仓库根目录运行任意静态服务器，例如：

```bash
python3 -m http.server 8000
```

然后打开 <http://localhost:8000>。
